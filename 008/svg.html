Displaying SVG using `<img>` tag

This is a common and easy way to include SVGs. It treats the SVG like a raster image.

How to use it:

<img src="path/to/your-file.svg" alt="Description of the SVG">

Important Considerations:

  • `src` attribute: Points to the path of your `.svg` file.
  • `alt` attribute: Essential for accessibility. Provides alternative text for screen readers and when the image fails to load.
  • Styling: You can control the `width` and `height` of the SVG using CSS on the `<img>` tag, just like with PNGs or JPEGs. However, you *cannot* style the internal elements of the SVG (like changing the color of a specific path) directly with CSS from your HTML file when it's embedded this way.
